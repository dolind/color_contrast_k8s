CPU Autoscaling Demo (C++ backend + TypeScript frontend)

Move the slider to simulate concurrent users. Higher load → higher CPU → Kubernetes scales pods.

- Inflight: is the number of requests that have been sent to the backend but not yet received a response. Analog to real usage: the number of users waiting for a response.
- Done: Completed requests. Real usage: users that have received a response.
- Errors: Requests that failed. Real usage: users that encountered an error. - Pods: Number of pods in the deployment.
