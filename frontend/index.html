- Inflight: is the number of requests that have been sent to the backend but not yet received a response. Analog to real usage: the number of users waiting for a response.
- Completed: Completed requests per second. Real usage: users that have received a response.
- Errors: Requests that failed. Real usage: users that encountered an error.
- Pods: Number of pods in the deployment.
